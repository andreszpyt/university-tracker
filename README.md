# Trilha Acadêmica — SI UFC Quixadá

Uma plataforma modular para acompanhamento, projeção e análise do progresso curricular dos estudantes de Sistemas de Informação.

## 🚀 Tecnologias

- **Frontend:** React + TypeScript + Vite
- **Estilização:** TailwindCSS (em breve)
- **Gráficos:** Chart.js / Recharts (em breve)
- **Backend/DB:** Node.js + MongoDB (em breve)

## 📦 Como rodar localmente

1. Clone o repositório.
2. Instale as dependências com `npm install`.
3. Inicie o servidor de desenvolvimento com `npm run dev`.
